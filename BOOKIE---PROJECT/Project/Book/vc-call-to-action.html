Important Note:

“Call To Action” Visual Composer add-on is optimised for “dark background”. To get the best result, please check these requirements:

  • Please choose “Visual Composer” page template from Page Attributes metabox
  • When adding Call To Action to a row, edit Row Settings – General tab – Row Stretch option, choose “Stretch row”
  • Again, edit Row Settings – Design Options tab:
    • add 0px to margin bottom
    • add #222222 for background color
    • add a dark background image and choose “Cover” from the dropdown
